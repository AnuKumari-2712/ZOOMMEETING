# 🎥 Zoom Elite Clone

A modern Zoom Clone built using **HTML, CSS, JavaScript, Node.js, Express.js, Socket.io, and WebRTC**. This project provides real-time video conferencing with a clean Zoom-inspired interface.

---

## 🚀 Features

- 🎥 Real-Time Video Calling
- 🎤 Mute / Unmute Microphone
- 📹 Camera On / Off
- 💻 Screen Sharing
- 💬 Live Chat
- ✋ Raise Hand
- 👥 Participant Counter
- 🆔 Join Meeting with Room ID
- 🚪 Leave Meeting
- 📱 Responsive Design
- 🌙 Modern Dark UI
- ⚡ Real-Time Communication using Socket.io
- 🌐 Peer-to-Peer Video Streaming using WebRTC

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/zoom-elite-clone.git
```

### Open project

```bash
cd zoom-elite-clone
```

### Install dependencies

```bash
npm install
```

### Start server

```bash
node server.js
```

### Open browser

```
http://localhost:5000
```
